# Train-Ticket-Booking-App
The Train Ticket Booking System is a console-based application developed in C language on Linux that simulates a basic railway reservation system. The application allows users to register, login, view available trains, book tickets, cancel tickets, and check booking details.
The system uses file handling for persistent data storage and linked lists for dynamic passenger management, providing a structured and modular implementation of a real-world booking system.
This project demonstrates concepts of data structures, file handling, modular programming, and system design in C.
